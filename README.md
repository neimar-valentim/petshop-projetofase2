Projeto Petshop Online – Fase 2Projeto desenvolvido para a disciplina Fundamentos de Sistemas Web, da PUC-RS.
Objetivo do ProjetoDar continuidade ao sistema web desenvolvido na fase 1, tornando a aplicação mais dinâmica, interativa e acessível por meio da utilização de CSS/Bootstrap e JavaScript.
Nesta fase, também foi implementado um formulário de cadastro do cliente e do pet, juntamente com um sistema de agendamento de serviços.
Funcionalidades Implementadas: 
Interface aprimorada com BootstrapEstilização adicional com CSSUso de JavaScript para interatividadeCarrossel de imagens na página inicialMensagem dinâmica baseada no horário do usuárioExibição de resumo do agendamento após envio do formulárioCadastro e AgendamentoO sistema conta com um formulário completo contendo:
Dados do clienteNomeCPFEndereçoTelefoneE-mailSexoDados do petNomeRaçaIdadePorteAgendamento de serviçosSeleção do serviço (banho, tosa ou ambos)Método de atendimento (tele-busca ou entrega no local)Data do agendamentoHorárioServiços adicionais (checkbox)AcessibilidadeForam aplicadas melhorias para garantir acessibilidade ao sistema:
Uso do atributo alt em todas as imagensAssociação de label aos campos do formulárioEstrutura semântica com header, main, section e footerUso de aria-live para feedback dinâmico ao usuárioTecnologias UtilizadasHTML5CSS3Bootstrap 5JavaScriptEstrutura do Projetoundefined
