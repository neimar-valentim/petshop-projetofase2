# Projeto Petshop Online – Fase 2

Projeto desenvolvido para a disciplina **Fundamentos de Sistemas Web – PUC-RS**.

## Objetivo

Evoluir o sistema da fase 1 utilizando **Bootstrap** e **JavaScript**, tornando o site mais dinâmico e interativo, além de incluir cadastro e agendamento.

## Funcionalidades

* Interface com Bootstrap
* Estilização com CSS
* Interatividade com JavaScript
* Carrossel de imagens
* Mensagem dinâmica por horário
* Resumo de agendamento

## Cadastro e Agendamento

Formulário com:

* Dados do cliente (nome, CPF, endereço, telefone, e-mail, sexo)
* Dados do pet (nome, raça, idade, porte)
* Agendamento (serviço, método, data, horário e adicionais)

## Acessibilidade

* Uso de `alt` nas imagens
* Labels nos campos
* Estrutura semântica

## Tecnologias

HTML5, CSS3, Bootstrap e JavaScript
